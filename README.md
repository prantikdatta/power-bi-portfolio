# 📊 Power BI Projects Portfolio

Welcome to my Power BI project repository.

This repository contains a collection of Power BI dashboards and analytics projects that I have built while learning and practicing data analysis, business intelligence, and data visualization.

The objective of these projects is to transform raw data into meaningful insights through interactive dashboards, data modeling, and effective storytelling.
