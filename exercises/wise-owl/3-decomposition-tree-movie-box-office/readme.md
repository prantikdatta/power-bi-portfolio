## Overview

This project demonstrates how to build and configure a **Power BI Decomposition Tree** to analyze movie box office revenue using multiple hierarchical dimensions.

The report enables interactive drill-down from overall box office revenue into:

- Certificate
- Genre
- Director

The project also demonstrates formatting options including node locking, connector customization, expandable nodes, and parent-relative scaling.

---

## Problem Statement

Create a decomposition tree visual showing movie box office takings.

Requirements:

- Analyze:
    - Box Office Dollars

- Explain By:
    - Certificate
    - Genre
    - Director

Expand the tree in this order:

1. Certificate
2. 12A
3. Comedy
4. Ivan Reitman

Then configure:

- Lock Certificate node
- Change connector colours
- Enable expand/collapse
- Scale bars relative to parent node

Finally save the report as:

```
Tree of life.pbix
```
