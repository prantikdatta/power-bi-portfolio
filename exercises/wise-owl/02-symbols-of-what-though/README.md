# Symbols of What Though - Power BI Practice

## Overview

This project is my solution to a Power BI practice exercise from Wise Owl.

The objective was to build an interactive report that explores skyscraper data using Power BI visualizations and drill-down functionality.

## Skills Practiced

* Creating Card visuals
* Building Bar Charts
* Applying conditional formatting
* Configuring drill-down hierarchies
* Formatting visuals
* Working with categorical data

## Report Features

* Card displaying the total number of skyscrapers
* Bar chart comparing skyscraper counts by country
* Conditional formatting based on building counts
* Drill-down from Country to City
* Data labels and customized titles
