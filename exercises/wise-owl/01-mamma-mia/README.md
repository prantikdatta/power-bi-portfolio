# Mamma Mia - Power BI Practice

## Overview

This project is my solution to a Power BI practice exercise from Wise Owl.

The objective was to:

- Import data from an Excel workbook
- Rename imported columns
- Hide unnecessary fields
- Create a simple table visual

## Skills Practiced

- Importing Excel data
- Power Query basics
- Renaming columns
- Hiding fields
- Building a Table visual
